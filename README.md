# Homework

上课期间值得保留的作业汇总。

## Embeded System
本科阶段嵌入式系统课程项目。实现一个无线视频监控系统，分为三个节点，视频采集节点A，中继节点B以及视频处理或监控节点C。系统设想的应用场景为，A节点是位于监控现场的设备，通过无线传输到实际的监控室中，但是由于距离太远而需要中继节点进行转发。节点C获得数据后进行数据的解码和显示。
